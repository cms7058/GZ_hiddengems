微信视频号信息

请填写可直接访问的图片 URL，系统会转存至 OSS。视频号播放链接不能作为封面图片。

微信视频号链接设置指南

  1. 用电脑打开视频号助手，用个人视频号绑定的微信扫码登录；在首页头像或账号信息附近查看视频号 ID，通常以 sph... 开头，填入 finder_user_name。
  2. feed_id：进入“内容管理 / 视频管理”，找到目标视频，点击右侧分享或更多，选择“复制视频 ID”。
  3. title：填写该视频标题，也可自定义。
  4. cover_url：右键点击封面图片并复制图片地址，然后点击“URL 转换”。
  5. 点击“添加视频”，确认下方出现视频信息和封面图片。

秘境图片/视频

图片支持 JPG、PNG、WebP、GIF，最大 2MB；视频支持 MP4、MOV、M4V，最大 8MB。仅图片可设为封面。
